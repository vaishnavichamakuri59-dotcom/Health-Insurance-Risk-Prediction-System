# Health Insurance Risk Prediction System

This project predicts health insurance charges based on customer attributes such as age, BMI, smoking status, and region.

## Features
- Machine Learning prediction using Random Forest
- Risk categorization (Low / Medium / High)
- Feature importance visualization
- Professional UI using Flask and Bootstrap
- Downloadable PDF report

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- Bootstrap
- ReportLab

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the application
python app.py

3. Open browser
http://127.0.0.1:5000
